Preguntas.

Incluir respuestas en algún punto del ejercicio práctico elegido por ustedes.

1) Dar un par de ejemplos de uso de etiquetas meta.
2) Nombrar al menos 2 atributos aplicables a la etiqueta HTML
3) Como se relaciona un header http con la etiqueta head de HTML?
4) Que es UTF8?
5) Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
6) Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?
7) Explicar cuando usarían etiquetas semánticas y con que propósito.
8) ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.
9) ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
10) ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

1) Algunos ejemplos de etiqueta meta son : Title , Keywords , author , Copyright , Charset , etc.

2) Lang(Se refiere al lenguaje) , base(especifica la dirección URL) . Ambos atributos se encuentran en desuso

3) No se relaciona, debido a que NO exite relacion entre un Header http y la etiqueta head de html.

4) UTF-8(UCS Transformation Format 8) es la codificacion de caracteres más común en la red. UTF-8 puede representar cuanquier carácter Unicode estandar.

5) El atributo que sirve para aplicar estilo es:
< style> < /style> se puede aplicar a practicamente todas las etiquetas.

6) Para que un enlace ubique el documento en una nueva pestaña se debe agregar el atributo " target " con el valor " _blank " Ejemplo:
< a href="Ubicacion del documento" target="_blank" > < /a>

7) En el encabezado del documento o titulo usaria Header dentro del mismo usaria el nav para crear una barra de navegación y el main para alojar el cuerpo del documento.

8)Los atributos que le aplicaria una tabla son:
Style: Para aplicarle estilos como por ejemplo color de fondo bordes etc.
Id: Para identificar la tabla en el documento.

9)La diferencia entre la etiqueta body de html y el body de Http es que el body de este ultimo puede contener datos en cualquier formato y el body html solo puede contener etiquetas html(elementos de flujo).

10) Los atributos que marcan la agrupacion de celdas de una tabla son:
Colspan: Avanza sobre las siguientes celdas hacia la derecha.
Rowspan: Avanza sobre las siguientes celdas hacia abajo.
Estos atributos se aplican a las etiquetas
